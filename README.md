# Java Fundamentals

A structured collection of Java programs covering fundamentals and core programming concepts.  
This repository contains programs and examples from my learning journey as an IT student at the University of Moratuwa.


## Topics Covered

1. **Basics**  
   - Hello World  
   - Variables and Data Types  
   - User Input  

2. **Conditions**  
   - If-Else Statements  
   - Nested If Statements  
   - Maximum Number Programs  
   - Grade Calculator  

3. **Loops**  
   - For Loop, While Loop, Do-While Loop  
   - Nested Loops  
   - Pattern Printing  

4. **Arrays**  
   - Single-Dimensional Arrays  
   - Multi-Dimensional Arrays  

5. **Methods**  
   - Defining and Calling Methods  
   - Method Overloading  

6. **Object-Oriented Programming (OOP)**  
   - Classes and Objects  
   - Constructors  
   - Inheritance  
   - Polymorphism  
   - Encapsulation
